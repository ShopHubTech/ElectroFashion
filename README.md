# ElectroFashion
Welcome to ElectroFashion, your go to destination for cutting-edge fashion and electronics! This open source E-commerce website is built to provide a sleek and secure platform for showcasing and selling the lastest wearable technology.

# Features
-Explore a curated collection of electrifying fashion items.
-Seamless and intuitive user interface for an enjoyable shopping experience.
-Secure and efficient payment processing.
-Resonsive design for consistent experience accross devices.
-Early customizable to fit your unique brand and product offerings.
